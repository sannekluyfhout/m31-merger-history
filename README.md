# M31 Merger History
Python code used to analyse the merger history of M31 analogues in constrained Local Group simulations.


This repository contains code and figures used for my bachelor thesis on the merger history of M31 in Local Group simulations.

## Contents

- `notebooks/`: Jupyter notebooks used for the analysis
- `scripts/`: Python scripts
- `figures/`: figures produced for the thesis
- `data/`: processed data files, if included
